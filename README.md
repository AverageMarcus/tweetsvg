# TweetSVG

Generate an SVG for a given Tweet ID

Available at https://tweet.cluster.fun/

## Features

* Provide the ID of a tweet and have it render as an SVG for use in an `<img>` tag.
* No JavaScript required

## Building from source

With Docker:

```sh
make docker-build
```

Standalone:

```sh
make build
```
